# 🎬 Análise de Filmes em Python

Este projeto é um sistema simples em Python para **gerenciar, analisar e buscar filmes**, utilizando leitura e escrita em arquivo `.csv`.

---

## 📌 Funcionalidades

O sistema possui um menu interativo com as seguintes opções:

### ➕ 1. Adicionar filmes
- Permite cadastrar um novo filme  
- Salva automaticamente no arquivo `filmes.csv`  
- Valida a nota (0 a 10)

---

### 📋 2. Listar filmes
Você pode visualizar os filmes de diferentes formas:

- Todos os filmes  
- Filtrar por gênero  
- Filmes recomendados (nota ≥ 7)

---

### 🔎 3. Buscar filme
- Pesquisa um filme pelo nome  
- Retorna gênero e nota caso exista  

---

### 📊 4. Análises
O sistema gera estatísticas automáticas:

- 🥇 Melhor filme (maior nota)  
- 🥶 Pior filme (menor nota)  
- 📂 Quantidade de filmes por gênero  
- 🏆 Top 3 filmes com maiores notas  

---

### ❌ 5. Sair
Encerra o programa.

---

## 🧠 Conceitos utilizados

- Manipulação de arquivos (`.csv`)  
- Listas e dicionários  
- Laços de repetição (`for`, `while`)  
- Condicionais (`if/elif/else`)  
- Funções  
- Ordenação com `sorted()` e `lambda`  
- Normalização de texto (remoção de acentos com `unicodedata`)
